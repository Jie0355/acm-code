# 直线划分平面
用 $n$ 条直线将一个平面划分成 F$(n)$ 个区域。  
假设当有 $n-1$ 条直线时，平面被划分成了 F$(n-1)$ 个区域。  
且此时 **MAX**{ F$(n-1)$ }。  
若添一条直线使得 **MAX**{ F$(n)$ }。  
就必须与其他直线相交且不能有同一交点，这样就可以得到 $n-1$ 个交点。  
这些交点将第 $n$ 条直线分为 $2$ 条射线和 $n-2$ 条线段。  
而每条射线和线段又将已有的区域一分为二。  
这样就多出了 $n$ 个区域。  
结论：F(n) = F(n - 1) + n  
数学：F(n) = n * (n + 1) / 2 + 1
